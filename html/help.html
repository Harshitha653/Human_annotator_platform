Help & Annotation Guidelines

How to Navigate

  • 1️⃣ Start from the Dashboard to choose or create a dataset.
  • 2️⃣ Click on Annotation Workspace to upload and label files.
  • 3️⃣ Use Save to update your progress.
  • 4️⃣ View insights in the Statistics page.

Annotation Guidelines

Follow these best practices while annotating:

  • ✔️ Read or view the content carefully before labeling.
  • ✔️ Choose consistent labels across similar data items.
  • ✔️ Use the “Neutral” label if unsure.
  • ✔️ Do not upload copyrighted or confidential material.
  • ✔️ Save your progress frequently.
